Описание
Используется для поиска по системе закупок.
Искусственное ограничение на выдачу: 500.

inn — поиск по ИНН поставщика;
kpp — поиск по КПП поставщика;
namesearch — полнотекстовый поиск по названию поставщика;
address — полнотекстовый поиск по местонахождению поставщика;
regioncode — поиск по региону поставщика (используются числовые коды регионов);
orgform — поиск по форме организации (используются коды организаций);
orgclass — "npo" или "university";
inblacklist — имеются ли нарушения у поставщика, параметры ["1", "true", "yes", "on", "y", "t"]
или ["0", "false", "no", "off", "n", "f"].

Специальные поля:
total - найдено записей;
page - страница в выдаче;
perpage - количество записей в одном запросе (max - 50).

Допускается сортировка выдачи по:
contractsCount — сортировка по количеству контрактов с параметрами [1, -1];
contractsSum — сортировка по сумме контрактов с параметрами [1, -1].

Примеры полнотекстового поиска по поставщикам
Поиск по региону:
http://openapi.clearspending.ru/restapi/v3/suppliers/search/?regioncode=35
Поиск по региону и форме организации:
http://openapi.clearspending.ru/restapi/v3/suppliers/search/?regioncode=35&orgform=i94
Поиск по региону и форме организации с убыванием по сумме контрактов: 
http://openapi.clearspending.ru/restapi/v3/suppliers/search/?regioncode=35&orgform=i94&sort=-contractsSum
