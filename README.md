# Программа для автоматизированного доступа к данным о государственных закупках.

Выполнение тестового задания для прохождения программы стажировки в лаборатории ТПУ "Финансовые технологии":
- Реализация прототипа программы для автоматизированного доступа к данным о государственных закупках. Можно использовать API Clearspending или любой аналогичный сервис.

Программа была разработана на языке C# с Windows Forms в качестве интерфейса пользователя. Данный прототип позволяет выбирать, какие именно данные пользователь желает получить из API сайта ГосЗатрат (контракты, гранты, заказчики, статистика), а также ввести значения параметров запроса.
